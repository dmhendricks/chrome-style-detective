Style Detective

%MANIFEST_VERSION%

Inspect CSS properties of any element on a page in real time — hover to reveal, freeze to study, copy to take with you.

How to Use

  1. Click the toolbar icon to enable Style Detective on the current page.
  2. Hover any element to inspect its computed CSS in the floating panel.
  3. Works on http, https, and local pages.

Keyboard Shortcuts

  • F Freeze / unfreeze the panel in place
  • C Copy the selected element's CSS to the clipboard
  • +/− Increase or decrease the panel font size
  • H Open this help page
  • Esc Disable the viewer

Privacy

Style Detective needs access to the current page so it can read styles. It neither collects nor transmits any of your private data.

Forked from miled/cssviewer and modernized.
