Style Detective 1.8.0

Style Detective is a simple CSS property viewer for Google Chrome originally made by Nicolas Huon as a Firefox addon.

Style Detective source code is publicly available on GitHub for anyone to help and contribute: https://github.com/miled/cssviewer.

How to use:

  • Style Detective shows the CSS parameters of any element in a web page. To enable/disable Style Detective, simply click the toolbar icon and then hover any element you want to inspect on the current page.
  • For quick usage you may use the available keyboard shortcuts:
    • F Freeze/Unfreeze the widget in place
    • C Show the selected element's CSS definition in a modal
    • Esc Disable the viewer
  • To display the CSS definition, open the Google Chrome console [F12] then right-click the element you're hovering:
    Style Detective console > element.simpleCssDefinition.

Note that the menu will only show up when you enable Style Detective and will only work on the page you're inspecting. Also, Style Detective will only work on http, https, and local pages.

Privacy:

In order to work properly, Style Detective requires permission to access your history and website data. However, Style Detective does not collect nor will ever transmit any of your private data.

How to uninstall:

  1. Click the puzzle-piece (Extensions) icon on the browser toolbar.
  2. Click Manage Extensions.
  3. Find the Style Detective extension, then click Remove.

Known Issues:

  • Style Detective will not work in any tabs opened before installing it, nor in the Chrome Web Store.
  • The style may break sometimes depending on the site you're surfing.

Style Detective automatically displays this page when the extension is updated to a new version.

This Chrome extension is a straight port of the Firefox CSSViewer add-on with minor enhancements. Credit goes to the original developer and contributors.
