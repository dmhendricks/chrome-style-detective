CSS Quick Viewer 1.8.0

CSS Quick Viewer is a simple CSS property viewer for Google chrome originally made by Nicolas Huon as a FireFox addon.

CSS Quick Viewer source code is publicly available on GitHub for anyone to help and contribute.
https://github.com/miled/cssviewer.


How to use :

  • CSS Quick Viewer shows the css parameters of any element in a web page. To enable/disable CSS Quick Viewer, simply click the toolbar icon and then hover any element on you want to inspect in current page.

  • For quick usage you may use the available keyboard shortcuts:

      [f] Freeze/Unfreeze the widget in place,
      [c] Shows selected element simpleCssDefinition on a modal, and
      [esc] Disable the viewer.

  • To display the CSS definition, open Google Chrome console [F12] then right click on the element you're hovering:

      CSS Quick Viewer console > element.simpleCssDefinition.

Note that the menu will only show up when you enable CSS Quick Viewer and will only work on the page you're inspecting. Also, CSS Quick Viewer will only work on http, https and local pages.

Privacy :

In order to work properly, CSS Quick Viewer requires the permission to access to your history and website data. However, CSS Quick Viewer do not collect nor will ever transmit any of your private data.

How to uninstall :

  1. Click the wrench icon on the browser toolbar.
  2. Click Tools.
  3. Select Extensions.
  4. Find the CSS Quick Viewer extension then click Uninstall.

Known Issues:

CSS Quick Viewer will not work in any opened tabs before installing it nor in the Google chrome store.
The style may broke sometimes depending on the site you are surfing.

CSS Quick Viewer automatically display this page when the extension is updated to a new version.

This chrome extension is a straight port of the Firefox CSSViewer add-on with minor enhancements. Credits goes to the original developer and contributors.
